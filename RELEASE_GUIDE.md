# ChaseGame Windows Release Guide

This project is distributed as a downloadable Windows build generated by GitHub Actions.

## For maintainers

1. Commit and push your code.
2. Create and push a version tag:
   - `git tag v0.1.1`
   - `git push origin v0.1.1`
3. Wait for workflow `Build Windows Exe` to complete.
4. Open the generated GitHub Release and verify attached files:
   - `ChaseGame.exe`
   - `ChaseGame-vX.Y.Z-windows.zip`
   - `SHA256SUMS.txt`

## For players

1. Open the latest GitHub Release.
2. Download `ChaseGame-vX.Y.Z-windows.zip`.
3. (Optional) verify checksum with `SHA256SUMS.txt`.
4. Extract the zip.
5. Run `ChaseGame.exe`.

## Notes

- Browser or Windows SmartScreen warnings can appear for unsigned executables.
- This is expected for indie builds without code-signing certificates.
- To reduce warnings further, use code signing in a future release.
